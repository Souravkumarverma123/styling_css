Features

Zero Build Step

Import and go. No webpack, no PostCSS, no config. Works in any project.

Hover & Focus

Real CSS pseudo-selectors. Hover effects, transitions, focus rings — all working.

Responsive

Mobile-first breakpoints. sm:, md:, lg:, xl: — all the responsive utilities.

Dark Mode

Automatic dark mode detection. Or use the dark: variant for manual control.

Tea Palette

26 hand-crafted tea colors. Masala chai, Darjeeling, Matcha, and more.

Arbitrary Values

Custom values when you need them. chai-bg-[#yourcolor], chai-p-[20px].
