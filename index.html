Why Lemon?

Zero Build Step

Import and go. No webpack, no PostCSS, no config. Works in any project — vanilla JS, React, Vue, Svelte.

Hover & Focus States

Real CSS pseudo-selectors. Hover effects, transitions, focus rings — all working out of the box.

Mobile-First Responsive

Responsive utilities with le-sm:, le-md:, le-lg:, le-xl: prefixes. Design for every screen size.

Dark Mode Support

Automatic dark mode detection with le-dark: variant for manual control. Your users' eyes will thank you.

Citrus Color Palette

26 hand-crafted colors. Lemon zest, orange peel, lime, grapefruit — plus full Tailwind spectrum.

Arbitrary Values

Custom values when you need them. le-bg-[#yourcolor], le-p-[20px], le-w-[calc(100%-40px)].
